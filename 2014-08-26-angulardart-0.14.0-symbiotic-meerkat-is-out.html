This release adds support for Polymer web components inside Angular components, using the new on-* and bind-* syntax. Take a look at example/web/paper.html for some material design examples.
                                                                                    
We have added instrumentation for the Web Tracing Framework, so you can visualize time spent in various Angular phases like scope digest, change detection, view creation (see ng_tracing_scopes.dart for a full list.) After installing the WTF plugin, you can jump straight into finding where your live app spends the most time. See our wiki for more information.
                
Finally, we’ve addressed bugs and made performance improvements for an even smoother developer experience.

For more information check out the changelog.
